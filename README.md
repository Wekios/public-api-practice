## Features

* Sass
* ES6
* Asset loading
* CSS Vendor prefixing
* Development server
* Sourcemaps
* Production optimizations

## Requirements

* [Node](https://nodejs.org) > 7.6

## Usage

Clone the repository

Install npm dependencies

```sh
 npm install 
```

To start the development server

```sh
npm start
```

To build for production

```sh
npm run build
```

To preview the production build
```sh
npm run preview
```
